Odoo and CVR Connection API

CVR Connection API App for Odoo.

This module provide feature to connect to CVR and fetch CVR number and other details.

Main Features:

  • Allow you to return CVR, Address of the Comapny.
  • Create partner record into Odoo.
  • You will have to supply your company name & get CVR, Address and other details.
